## 📚 وصف المشروع

موقع ويب شامل وحديث لمدرسة بنايوس الثانوية المشتركة، يوفر للمدرسة والطلاب والمعلمين وأولياء الأمور واجهة سهلة الاستخدام للوصول إلى جميع المعلومات المهمة.

### ✨ المميزات الرئيسية

- 🔐 **نظام تسجيل دخول وتسجيل جديد** - نظام مصادقة متكامل
- 👥 **قوائم تفاعلية** - قوائم الطلاب والمعلمين مع وظيفة البحث
- 📅 **جداول الحصص** - جداول دراسية لجميع الفصول مع البحث والطباعة
- 🏆 **لوحة الشرف** - عرض أوائل الطلاب
- 🌙 **الوضع الليلي** - دعم الوضع المظلم
- 📱 **تصميم متجاوب** - يعمل على جميع الأجهزة
- 🎨 **تصميم عصري** - رسوم متحركة وتأثيرات بصرية جذابة
- 💾 **تخزين محلي** - حفظ البيانات بدون قاعدة بيانات خارجية
- 🔗 **روابط سريعة** - روابط للمنصات التعليمية وتواصل وسائل التواصل الاجتماعي

## 🚀 البدء السريع

### المتطلبات

- متصفح حديث (Chrome, Firefox, Safari, Edge)
- لا حاجة لأي ملحقات أو خوادم

### التثبيت

#### للاستخدام الشخصي (بدون خادم):
1. **استنساخ المشروع**:
```bash
git clone <repository-url>
cd school-website
```

2. **تشغيل الموقع**:
   
   **الخيار 1**: افتح `index.html` مباشرة في المتصفح
   
   **الخيار 2**: استخدم Live Server في VS Code
   - افتح المشروع في VS Code
   - اضغط على `Go Live` أو استخدام الاكستنشن
   - الموقع سيعمل على `http://127.0.0.1:5501`

#### للوصول من أجهزة أخرى (مع خادم Node.js):

**📖 راجع ملف [`SERVER_SETUP.md`](SERVER_SETUP.md:1) للتعليمات الكاملة**

الملخص السريع:
```bash
# تثبيت التبعيات
npm install

# تشغيل الخادم
npm start

# أو للتطوير (مع إعادة التحميل التلقائي)
npm run dev
```

بعد التشغيل، الموقع سيكون متاحاً على:
- **الجهاز المحلي**: http://localhost:3000
- **الأجهزة الأخرى على نفس الشبكة**: http://192.168.x.x:3000 (استخدم عنوان IP الخاص بك)

## 🔧 التخصيص

### تغيير الألوان
في [`style.css`](style.css:4)، ابحث عن متغيرات CSS:

```css
:root {
    --primary-color: #1a73e8;
    --secondary-color: #34a853;
    --accent-color: #0d6efd;
    /* ... */
}
```

### إضافة مستخدمين جدد
يمكنك إضافة مستخدمين عبر:
1. **التسجيل عبر موقع الويب**: استخدم صفحة [`register.html`](register.html:1)
2. **تصدير/استيراد JSON**: استخدم دوال JavaScript في [auth.js](auth.js:1)

```javascript
// تصدير المستخدمين
const jsonData = exportUsersData();

// استيراد المستخدمين
importUsersData(jsonString);
```

// الحصول على المستخدم الحالي
